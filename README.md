# IPR Platform

A blockchain-powered Intellectual Property Rights management platform built with Next.js, Supabase, and Solana.

## Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm
- Supabase account
- Solana wallet (Phantom recommended)
- Git
